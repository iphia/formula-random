# 공식 랜덤 암기

설명과 공식을(또는 개념과 정답을) 단계적으로 확인하며 외우기 위한 간단한 정적 웹앱입니다.  
처음에는 **설명(힌트)** 을 먼저 보여주고, 같은 항목을 다시 누르면 **공식/정답(수식)** 을 보여주는 방식이라 “이해 → 암기” 흐름으로 사용하기 좋습니다.

또한 본 프로젝트는 **ChatGPT의 도움을 받아 제작**되었습니다.

---

## 주요 사용 흐름

- 화면(스테이지)을 누르면 **설명**이 먼저 표시됩니다.
- 한 번 더 누르면 **공식/정답(KaTeX 수식)** 이 표시됩니다.
- 다시 누르면 **다음 항목(설명부터)** 으로 넘어갑니다.

> 요약: **설명 → (다시 누름) 공식/정답 → (다시 누름) 다음(설명부터)**

---

## 기능

- 설명/공식(정답) 2단계 표시(토글 방식)
- 랜덤 출제(한 바퀴 완료 후 다시 랜덤 구성)
- 항목 관리: 추가/수정/삭제
- 학습 관리: 포함/제외(=반복에서 제외) 목록
- 데이터 로컬 저장(LocalStorage)
- KaTeX 수식 렌더링 지원
- **백업(json다운로드) / 복원(json업로드) 지원**

---

## 커스터마이징

- 항목 데이터(설명/공식)는 `app.js`의 데이터/저장 로직을 통해 추가 및 관리됩니다.
- 수식은 KaTeX 문법을 사용하므로, 일반 텍스트뿐 아니라 수식 기반 콘텐츠에도 적합합니다.
