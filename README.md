# LaTeX 공식 랜덤 암기 웹앱

LaTeX 수식을 랜덤으로 보여주고, 외운 공식은 제외하면서 암기할 수 있는 간단한 웹앱입니다.

- 화면을 탭하면 다음 공식이 랜덤으로 표시됩니다.
- **미암기 목록 / 제외 목록**을 3열 그리드로 확인할 수 있습니다.
- 우하단 메뉴에서 **공식 추가 / 백업 / 복원**을 지원합니다.
- 패널에서 항목을 **길게 누르면 삭제**할 수 있습니다.

> 이 프로젝트는 **ChatGPT의 도움을 받아** 제작되었습니다.

---

## 주요 기능

### 랜덤 표시
- 앱 실행 시 미암기(=제외되지 않은) 공식 중 하나가 랜덤으로 표시됩니다.
- 화면을 탭할 때마다 다음 공식이 랜덤으로 표시됩니다.

### 외운 공식 제외
- 상단 중앙 **“이 공식 외웠음(제외)”** 버튼으로 현재 공식을 제외 목록으로 이동할 수 있습니다.

### 목록 보기
- 상단 왼쪽 **미암기**: 아직 제외되지 않은 공식 목록 (탭하면 바로 보기)
- 상단 오른쪽 **제외**: 제외된 공식 목록 (탭하면 다시 포함)

### 공식 추가
- 우하단 `+` → **공식 추가**
- 설명(선택) + LaTeX 코드 입력 후 저장
- 저장된 데이터는 브라우저 `localStorage`에 보관됩니다.

### 백업 / 복원
- 우하단 `+` → **백업**: 현재 데이터를 JSON 파일로 다운로드합니다.
- 우하단 `+` → **복원**: JSON 파일 업로드로 데이터 전체를 복원합니다.
- 기기 간 이동은 “백업 → 다른 기기에서 복원” 방식으로 진행합니다.

### 삭제
- 미암기/제외 패널에서 항목을 **길게 누르면 삭제 확인창**이 표시됩니다.
- 삭제한 공식은 백업 파일이 없으면 복구할 수 없습니다.

## 사용 방법

1. 이 저장소를 클론하거나 파일을 내려받습니다.
2. `index.html`을 브라우저에서 열어 테스트합니다.
3. GitHub Pages로 호스팅하면 모바일 홈화면 웹앱으로도 사용할 수 있습니다.

---

## GitHub Pages 배포

1. GitHub에 이 프로젝트를 업로드합니다.
2. `Settings → Pages`로 이동합니다.
3. **Deploy from a branch**를 선택합니다.
4. Branch: `main`, Folder: `/ (root)`를 선택한 뒤 저장합니다.

Pages 주소로 접속하면 앱이 동작합니다.

---

## 주의사항

- 데이터는 **각 기기의 브라우저 로컬(localStorage)** 에 저장됩니다.  
  → 다른 기기에서 동일 데이터를 사용하려면 **백업/복원**이 필요합니다.
- LaTeX 렌더링은 **KaTeX(CDN)** 을 사용합니다.
